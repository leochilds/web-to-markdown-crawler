# Security Policy

## Supported Versions

Only the latest release is actively supported with security updates.

| Version | Supported |
| ------- | --------- |
| latest  | ✅        |
| older   | ❌        |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report vulnerabilities privately via [GitHub's private vulnerability reporting](https://github.com/leochilds/web-to-markdown-crawler/security/advisories/new).

You can expect an acknowledgement within 48 hours and a resolution or status update within 7 days.
